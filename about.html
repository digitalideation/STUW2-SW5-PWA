This is a second page to demonstrate that navigation works offline too.

How does offline navigation work?

The service worker caches all three pages (index.html, apis.html, about.html) when it installs. When you click a link the browser makes a normal HTTP request, but the service worker intercepts it and responds with the cached copy — no network needed.

What is a PWA?

A Progressive Web App is a regular website enhanced with a manifest (so it can be installed) and a service worker (so it works offline). That's really all there is to it.
